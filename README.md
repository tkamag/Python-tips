## JSON Syntax
The syntax of JSON is straightforward. It is built on top of two simple structures:

* A collection of ``name/value pairs`` (in Python, they are represented as a Python dictionary).
````JSON
{
    "name": "James",
    "gender": "male"
    "age": 32
}
````
* An ordered list of values.
